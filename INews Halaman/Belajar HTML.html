Belajar Dasar HTML

Penjelasan:

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat dan merancang halaman web. HTML digunakan untuk menata struktur konten di web, seperti teks, gambar, dan tautan, sehingga dapat dilihat dan diakses oleh pengguna internet melalui peramban (browser) seperti Google Chrome atau Mozilla Firefox.


HTML singkat dari HyperText Markup Language adalah bahasa dasar yang digunakan untuk membuat halaman web. Pikirkan HTML sebagai "kerangka" atau "tulang" dari sebuah halaman web. Ia berfungsi untuk mengatur bagaimana setiap bagian dari sebuah situs web terlihat dan bekerja. Dengan HTML, kita bisa menyusun teks, gambar, video, dan elemen lainnya menjadi satu halaman yang terstruktur rapi dan mudah dibaca.

Cara kerja HTML

Cara Kerja HTML
HTML terdiri dari kumpulan kode yang dikenal sebagai tag. Setiap tag memberitahu peramban apa yang harus dilakukan dengan konten tertentu. Contohnya, <h1> digunakan untuk membuat judul utama, <p>untuk paragraf, dan <img> untuk menampilkan gambar. Jika kita ingin membuat sebuah paragraf, kita membungkus teks dengan tag <p></p>.

Contoh kodenya:

<h1>Selamat Datang di Situs Kami </h1>
<p>Ini adalah paragraf pertama di situs kami. </p>
<img src="gambar.jpg" alt="Gambar Contoh">

Struktur Dasar HTML ( Wajib )

<html>
<head> <title>Judul</title>
</head>
<body>
</body>
</html>

Contoh Hasilnya

<html>
<head> <title>Judul</title>
</head>
<body>
<h1>Selamat Datang di Situs Kami </h1>
<p>Ini adalah paragraf pertama di situs kami. </p>
<img src="gambar.jpg" alt="Gambar Contoh">
</body>
</html>

Kode di atas akan menampilkan judul besar "Selamat Datang di Situs Kami" diikuti dengan teks paragraf dan gambar.

Mengapa HTML Penting?

HTML sangat penting karena ia adalah fondasi dasar dari setiap situs web yang kita kunjungi. Dengan menggunakan HTML, kita bisa mengatur berbagai komponen pada halaman agar terlihat baik di browser. HTML juga sangat fleksibel dan mudah dipelajari, sehingga orang yang baru mengenal pembuatan web pun bisa langsung mencoba membuat halaman sederhana.

HTML untuk Pemula

Untuk pemula, mempelajari HTML bisa seperti belajar bahasa baru. Namun, jangan khawatir, HTML sangat logis dan tag-tag yang digunakannya memiliki nama yang mudah dimengerti, seperti <header> untuk bagian atas halaman, <footer> untuk bagian bawah, dan <body> untuk bagian isi utama.

Dengan hanya beberapa tag dasar, kita bisa mulai membuat situs web yang menarik. Untuk pemula, belajar HTML memberikan landasan kuat sebelum mempelajari CSS (untuk gaya tampilan) dan JavaScript (untuk interaktivitas).
